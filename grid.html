Button

Use Idli’s custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Usage

Script tag

  • Put a script tag similar to this <script src='https://unpkg.com/@idli/idli-button@0.5.0/dist/idli-button.js'></script> in the head of your index.html
  • Then you can use the element anywhere in your template, JSX, html etc 
                <!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
  <title>JS Bin</title>
  <script src='https://unpkg.com/@idli/idli-button@0.5.0/dist/idli-button.js'></script> 
</head>
<body>
  <idli-button size="lg" variant="primary">Click me</idli-button>
</body>
</html>

Sizes

Fancy larger or smaller buttons? Add size attribute for additional sizes.

Large Button Large Button 
                  <idli-button size="lg">Large Button</idli-button>
                  <idli-button variant="primary" size="lg">Large Button</idli-button>
Small Button Small Button 
                    <idli-button size="sm">Small Button</idli-button>
                    <idli-button variant="primary" size="sm">Small Button</idli-button>

Disabled state

Make buttons look inactive by adding the disabled boolean attribute to any <button> element.

Disabled Button Disabled Dashed Button Disabled Link 
                    <idli-button disabled>Disabled Button</idli-button>
                    <idli-button variant="dashed" disabled="true">Disabled Dashed Button</idli-button>
                    <idli-button variant="link" disabled="true">Disabled link</idli-button>
