{{ page.title }}

Use Idli’s custom input component for capturing input in forms, dialogs, and more with support for multiple sizes, variants and more.

Usage

Script tag

  • Put a script tag similar to this <script src='https://unpkg.com/@idli/idli-input@{{ page.version }}/dist/idli-input.js'></script> in the head of your index.html
  • Then you can use the element anywhere in your template, JSX, html etc 
                <!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
  <title>{{ page.title }}</title>
  <script src='https://unpkg.com/@idli/idli-input@{{ page.version }}/dist/idli-input.js'></script>
</head>
<body>
   <idli-input label="Name" placeholder="John Snow"></idli-input>
   <idli-input variant="dashed" label="Name" placeholder="John Snow"></idli-input>
</body>
</html>

Examples

Here are several predefined button styles, each serving its own semantic purpose.

                 <idli-input label="Name" placeholder="John Snow"></idli-input>
                 <idli-input variant="dashed" label="Name" placeholder="John Snow"></idli-input>

Inline

Display label and field on same line.

                  <idli-input inline label="Address" placeholder="John Snow"></idli-input>

Sizes

Fancy larger or smaller input? Add size attribute for additional sizes.

                  <idli-input size="lg" label="Large Input"></idli-input>
                  <idli-input variant="dashed" size="lg" label="Large Input"></idli-input>
                  
                
                    <idli-input size="sm" label="Small Input"></idli-input>
                    <idli-input variant="dashed" size="sm" label="Small Input"></idli-input>

Events

On click the button a CustomEvent buttonClick is triggered by the element.

Note: Check browser console for output.

                    <script>
document.querySelectorAll('#doc-events idli-input').forEach(function(el) {
    el.addEventListener('inputChange', function(event) {
        console.log("Input changed to :: " + event.detail.newValue + " - from :: " + event.detail.oldValue);
    })
});
</script>
